---
name: No "original analysis" references
description: When updating write-ups with corrections, just state the corrected facts — don't reference what the original draft said
type: feedback
---

When updating experiment write-ups or log entries with corrections, just state the corrected facts directly. Don't add phrases like "the original analysis reported X" or "the original draft had this backwards." We don't care about the history of the draft — just what the final result is.

**Why:** The write-ups should read as clean final documents. Tracking what was wrong before is noise — it's already in the reviewer files if anyone needs it.

**How to apply:** When editing any research_log/ markdown or LOG.md entry, write corrections as if they were always there. Remove any existing "original draft said X" language too.
